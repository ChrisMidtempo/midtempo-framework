# Contributing

Thank you for your interest in this project.

## Bug Reports and Discussion

Bug reports are welcome. Please use the [issue templates](https://github.com/ChrisMidtempo/agentic-framework/issues/new/choose) to ensure your report includes the information needed to investigate.

For questions and general discussion, please use [GitHub Discussions](https://github.com/ChrisMidtempo/agentic-framework/discussions).

## Code Contributions

This project is not currently accepting code contributions. Pull requests opened without a prior approved issue will be closed without review.

If you have an idea for a change, please open an issue to discuss it first.
